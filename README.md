# 🛒 E-Commerce Web Application (ASP.NET Core MVC)

## 📌 Overview

A full-featured E-Commerce web application built using ASP.NET Core MVC.
The project demonstrates real-world concepts such as authentication, authorization, product management, shopping cart, and order processing.

---

## 🚀 Features

### 👤 User Features

* Browse products with categories
* View product details
* Add/remove products to cart
* Manage cart (increase/decrease quantity)
* Checkout and place orders
* View order history

---

### 🛠️ Admin Features

* Full CRUD operations for products and categories
* Upload and manage product images
* Manage orders and update order status (Pending, Processing, Completed)
* Role-based access control (Admin/User)

---

## 🔐 Authentication & Authorization

* ASP.NET Core Identity integration
* User registration and login system
* Role-based authorization (Admin & User)
* Protected admin routes

---

## 🛒 Shopping Cart

* Session-based cart system
* Add, remove, and update product quantities
* Dynamic total price calculation

---

## 📦 Order System

* Convert cart into orders
* Store order details in database
* Order history per user
* Admin order management dashboard

---

## 🧱 Technologies Used

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* ASP.NET Identity
* Bootstrap (UI)
* Session State

## ⚙️ Setup Instructions

1. Clone the repository
2. Update connection string in `appsettings.json`
3. Run:

   ```bash
   Update-Database
   ```
4. Run the project
